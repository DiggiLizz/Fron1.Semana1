Sekhmet - Catalogo de Anime y Videojuegos
Este proyecto es para la evaluacion formativa de la semana 4. Es un sitio web responsivo hecho con Bootstrap 5.

Cumplimiento de la pauta de evaluacion
Segun los requisitos de Duoc UC, el proyecto incluye:

Estructura Semantica (Criterio 1): Uso de etiquetas HTML5 como nav, main, section y footer para que el sitio sea accesible.

Navegacion Funcional (Criterio 2): Navbar fija (sticky-top) con enlaces que funcionan para ir a inicio, videojuegos y anime.

Sistema de Grids (Criterio 1 y 6): Uso de contenedores (container), filas (row) y columnas (col) de Bootstrap 5 para que el diseño sea responsivo en celulares, tablets y pc.

Componentes Bootstrap 5 (Criterios 3, 4 y 5):

Carrusel con transiciones automaticas.

Cards para mostrar las reseñas de forma ordenada.

Modales para ver los detalles, opiniones y videos de cada titulo.

Codigo y Diseño (Criterio 6): Codigo limpio, ordenado y con comentarios. Se uso el modo oscuro de Bootstrap 5.

Publicacion (Criterio 7): Proyecto subido a GitHub y publicado en la rama gh-pages.
